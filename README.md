# Blossom Institute – Frontend

Frontend web application for an educational management platform, built with Next.js and TypeScript. It integrates with a .NET backend API and provides role-based interfaces for administrators, teachers, and students.

## ⚠️ Status
This project is currently under active development. Features and structure may change.

## 🛠 Tech Stack
- ⚡ Next.js (App Router)
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🧩 Radix UI

## 🚀 Features (in progress)
- 🔐 Authentication (JWT-based)
- 👥 Role-based dashboards (Admin, Teacher, Student)
- 📚 Course and user management
- 📝 Assignments and submissions
